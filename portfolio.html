a tool to visualize store inventory

a new way to visualize instant, crowdsourced data. Functions by taking Azure Maps API, showing local stores to users, if they're within 1000 yards, have the ability to set store inventory of toilet paper. The data is then recorded on MySQL, then retrieved to overwrite map with a new icon depicting inventory level.

  • database stores inventory once stores are registered by user interaction
  • Built using Node.js, API protocols
  • Utilizes PassportJS, Router, ExpressJS.
  • originally created on GitHub.
  • built with Microsoft Azure Maps
